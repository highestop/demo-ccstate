# demo-ccstate

> _[CCState](https://github.com/e7h4n/ccstate) is a modern signals-based state management library that elegantly implements async computed and read-write capability isolation based on [signal](https://developer.mozilla.org/en-US/docs/Web/API/AbortSignal) features, making it suitable for medium to large web applications._
>
> _The name of CCState comes from three basic types: `Computed`, `Command`, and `State`._

This repo includes receipts and demos (possibly some good practice in personal use) for this lib.
